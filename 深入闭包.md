#### 定义

《JS忍者秘籍》中对闭包的定义为：

> 闭包实际上是一个作用域，在创建时允许其自身的变量和函数访问其自身之外的变量时创建的作用域。

所以，闭包是由两部分组成的:
> 闭包 = 函数 + 函数能够访问的自有变量

看个例子：
```
var outerValue = 1; // 全局作用域声明一个变量

function outerFunction() { // 在全局作用域中声明一个函数
    console.log(outerValue); // 1
}

outerFunction(); // 执行该函数
```
该函数是能够访问到自身之外的变量outerValue的，
